ESP8266 Web Server with Sensor Data

This webpage displays the values coming from the Photocell (Light Dependent Resistor) in real-time! without actually refreshing the page. This is done by implementing Asynchronous JavaScript and XML or AJAX to the ESP8266 Web Server.

LDR State values are as follows:

512 and above
- Darkest

256 - 512
- Darker

256 and below
- Darker
